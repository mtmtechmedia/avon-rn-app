## How to Start

```
git clone https://github.com/Avonee/TaipeiActivityGuideList_RN_Interview.git
```
```
npm i
```
```
npx pod-install ios
```
```
npx react-native start
```
open the new terminal inside the project
```
npx react-native run-ios
```
